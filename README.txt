1) Закидывать исходники в папку files/txt (имя исходных файлов должно быть порядковым числом).
2) Запускать один из скриптов из папки scripts/

Описание скриптов:
___________________________________________________________________________

format_files:

Обработает все файлы из папки files/txt и создает из них csv файлы.
Для каждого файла будут созданы все версии. Например для файла 1.txt будут созданы файлы 1v1.csv, 1v2.csv, ...
Все csv файлы будут сохранены в папку files/csv

Запускается командой: 
node format_files.js --c=count,
где count - название последнего файла (цифра)
___________________________________________________________________________

proccessing_files:

Обработает все файлы из папки files/txt и создает для них csv файл (по умолчанию)
в корне папки files/ с именем в следующем формате:
"(режим)_(значение напряжения)_results_proccessing_(время)_(дата)"

Режимы: default или inverse (Если "R v1" > "R v2", то они меняются местами)

Запускается командой:
node format_files.js --c=count --u=u --m=invert --format=format,
где count - название последнего файла (цифра)
    u - значение напряжения для которого выполняются расчеты
    m - режим, по умолчанию default.
    format - формат в котором сохраняется файл (csv или txt)
